National Exposure - Port Wars

You can use this interactive visualization to explore the battle between encrypted & unencrypted services.

The default view is showing plaintext (HTTP) vs encrypted (HTTPS) nodes. The bars are sorted by regions with the worst unencrypted vs encrypted makeup to the best.

The table below the chart shows the counts and percentages for each country.

Select a different pair from the popup menu to change the graph & table.
